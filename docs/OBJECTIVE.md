# Objective

## Primary Objective (MVP)
Demonstrate a pose-first pipeline that runs DeepLabCut SuperAnimal Quadruped on a known-good horse video and then on Jack's stable phone footage, producing overlay, metrics, and a basic activity report.

## Secondary Objectives
- Establish reproducible environment + CLI runner
- Define practical capture protocol for stable footage
- Create a go/no-go signal for fine-tuning based on likelihood/jitter/visual QC

## Explicit Non-Objectives
- Emotion, stress, pain, or health diagnosis classification
- Realtime mobile deployment
- Multi-horse tracking in a single clip
